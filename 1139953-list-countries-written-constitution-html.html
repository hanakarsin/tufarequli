List of countries with written constitution

Publish date: 2024-03-28

A written constitution is a formal document which outlines the nature of the constitutional settlement, the set of rules that govern the political system and the rights of citizens and governments in a codified form.

Almost every country has written constitution. Thus, we should focus on how many countries have unwritten constitution.

What is an unwritten constitution?

An unwritten constitution is a type of constitution where the constitutional rules are not in a codified form. That is, the rules are listed in several documents and not in a single document like written constitution.

Below is a list of countries with unwritten constitution:

1. Israel

2. Canada

3. Saudi Arabia

4. New Zealand
